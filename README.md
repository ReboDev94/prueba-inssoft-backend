# Inssoft - Backend

## Getting started

1.-Instalar dependencies

```sh
$ npm install
```

2.-Copiar environments

```sh
cp .env.example .env
```

3.- Levantar base de datos- es necesario usar docker

```sh
$ docker-compose up -d
```

4.- Levantar en modo de desarrollo

```sh
# watch mode
$ npm run start:dev
```
